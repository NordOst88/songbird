# Songbird

---

Songbird is a SPA quiz for recognizing birds by their voices.

### Demo: [https://nordost88-songbird.netlify.app](https://nordost88-songbird.netlify.app)

---

## Installation and Usage

---

To install: `npm install`
To start: `npm start`

## Tech stack

---

- **React**
- **Redux**
- **Redux-Toolkit**
- **SASS**
- **Bootstrap**
